# HerArmor - Tamil Nadu Crime Safety Map

An interactive Google Maps application showing crime levels and safety times across Tamil Nadu locations.

## Features

- **Crime Level Visualization**: Markers colored by crime severity (Green: Low, Yellow: Moderate, Red: High)
- **Safety Time Information**: Click markers to view safe/unsafe time slots
- **Detailed Data**: Includes rape cases, women cases, and scores for Vellore district
- **Responsive Design**: Works on desktop and mobile

## How to Use

1. Open `index.html` in a web browser
2. For local development, run `node server.js` and visit `http://localhost:8000`
3. Click on markers to see location details and safety times

## Data Sources

- Vellore crime data with detailed statistics
- General crime scores for other Tamil Nadu cities
- Safety times based on crime patterns

## Technologies Used

- Google Maps API
- JavaScript (ES6)
- JSON data format
- Node.js for local server
